DN42 WHOIS Explorer - API

WHOIS API Documentation

How to use the API?

Access DN42 registry data programmatically with simple HTTP requests:

GET http://whois.asmltfr.dn42/api/{type}/{object}

GET http://whois.asmltfr.dn42/api/inet6num/fd8b:dc6c:8653::/48 # Inet6num object
GET http://whois.asmltfr.dn42/api/route6/fd8b:dc6c:8653::/48 # Route6 object
GET http://whois.asmltfr.dn42/api/inetnum/172.20.18.240/28 # Inetnum object
GET http://whois.asmltfr.dn42/api/route/172.20.18.240/28 # Route object
GET http://whois.asmltfr.dn42/api/domain/asmltfr.dn42 # Domain objects
GET http://whois.asmltfr.dn42/api/aut-num/4242422412 # Aut-Num objects

Supported Object Types

The API supports the following DN42 registry object types:

  • inet6num - IPv6 address allocations
  • route6 - IPv6 routing information
  • inetnum - IPv4 address allocations
  • route - IPv4 routing information
  • domain - DNS domain information
  • aut-num - Autonomous System information

Response Format

All API responses return plain text data directly from the DN42 registry in standard WHOIS format. The data is served raw and live with no caching delays.
