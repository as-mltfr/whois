WHOIS Socket Documentation

How to use the WHOIS server?

Connect to our WHOIS server using any standard WHOIS client or netcat. The server listens on port 43 and accepts queries for DN42 registry objects.

whois.example.dn42:43

Supported Query Types

  • IPv6 prefixes (e.g., fd8b:dc6c:8653::/48)
  • IPv4 prefixes (e.g., 172.20.18.240/28)
  • Domain names (e.g., asmltfr.dn42)
  • Subdomains (e.g., subdomain.asmltfr.dn42)
  • AS numbers (e.g., as4242422412)

Query Examples

Here are some examples of valid queries you can send to the WHOIS server:

fd8b:dc6c:8653::/48
172.20.18.240/28
asmltfr.dn42
subdomain.asmltfr.dn42
as4242422412

Command Line Usage

You can query the WHOIS server directly from your terminal:

# Using whois command
whois -h whois.example.dn42 fd8b:dc6c:8653::/48

# Using netcat
echo "fd8b:dc6c:8653::/48" | nc whois.example.dn42 43

# Using telnet
telnet whois.example.dn42 43

Response Format

The WHOIS server returns standard WHOIS-formatted data directly from the DN42 registry. All data is served raw and live with no caching delays.
